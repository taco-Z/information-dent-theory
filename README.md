# Information Dent Theory

## ■ Overview

Information Dent Theory proposes that integrated structures of information generate potentials (depressions) in space, and these potentials manifest as physical phenomena such as gravity.
As the universe expands, it continuously creates unstructured regions, which are then filled through increasing information integration and complexity.

---

## ■ Fundamental Definitions

### ● Node

A unit that stores, processes, and integrates information
(e.g., neurons, electronic states, storage media)

### ● Interdependence

A relationship where changes in one node influence other nodes and the overall system

### ● Integrated Complexity

A structure formed by interdependent nodes that cannot be decomposed without loss of function

### ● Potential (Depression)

The influence exerted on space by integrated information structures

---

## ■ Core Hypotheses

1. Properties emerge from relationships, not from individual nodes
2. Gravity is one observable effect of information structure
3. Black holes represent a critical state of integrated information
4. The universe maintains a continuous complexity loop through expansion
